Ejemplo de Operadores Lógicos

Para que puedas practicar y aprender más sobre los operadores lógicos y cómo funcionan, elabora el siguiente ejercicio:

Ejercicio: Escribe un programa que verifique si una persona puede votar. La persona debe tener al menos 18 años y ser ciudadano.

Ejemplo de código:


edad = 20
es_ciudadano = True

if edad >= 18 and es_ciudadano:
    print("Puedes votar.")
else:
    print("No puedes votar.")

Ahora modifica el programa para incluir un operador lógico or que permita votar si la persona tiene un permiso especial, incluso si no cumple con las condiciones anteriores.

Ejemplo de código:


edad = 16
es_ciudadano = False
permiso_especial = True

if (edad >= 18 and es_ciudadano) or permiso_especial:
    print("Puedes votar.")
else:
    print("No puedes votar.")
