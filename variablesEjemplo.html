Ejercicio de variables

Crea un programa en Python que almacene en tres variables la información de una persona: su nombre, su edad y la ciudad donde vive. Asigna valores a cada una de estas variables (puedes inventarlos). Luego, utiliza la función print() para mostrar un mensaje que combine esas variables, siguiendo este formato: "Hola [nombre], tienes [edad] años y vives en [ciudad].". Asegúrate de que las variables se integren correctamente dentro del mensaje para que los datos se muestren de forma clara.

Ejemplo de código:


nombre = "Juan"
edad = 25
ciudad = "Madrid"
print(f"Hola {nombre}, tienes {edad} años y vives en {ciudad}.")

Ahora, modifica el código para incluir una variable adicional que almacene la nacionalidad de la persona. Asegúrate de que el mensaje final incluya esta nueva variable.

Ejemplo de código modificado:


        
print(f"Hola {nombre}, tienes {edad} años y eres de {nacionalidad}.")

Recuerda que puedes cambiar los valores de las variables para probar diferentes resultados. ¡Diviértete programando!
