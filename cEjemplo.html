Ejercicio de condicionales

Para que puedas practicar un poco mas y puedas aprender sobre los condicionales Desarrolla el siguiente ejercicio

Ejercicio: Escribe un programa que le pida al usuario un número y le diga si es par o impar.

Ejemplo de código:

        numero = int(input("Introduce un número: "))
        if numero % 2 == 0:
            print(f"{numero} es par.")
        else:
            print(f"{numero} es impar.")
