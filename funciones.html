Funciones

💡 ¿Qué son las funciones?

Una función es un bloque de código que solo se ejecuta cuando la llamas. Son geniales para reutilizar código y mantener tu programa organizado. En pocas palabras es "Define una función una vez y úsala muchas veces".

Para poder definir una función debes seguir los siguientes pasos:

  • Definir el nombre de la función con la palabra clave def.
  • Especificar el nombre de la función.
  • Incluir paréntesis con parámetros (si es necesario).
  • Escribir el código indentado que se ejecutará.

        def adivina_edad_dragon():
        edad_real = 587
        intento = int(input("¿Cuántos años crees que tiene el dragón? 🐉: "))
    
        if intento == edad_real:
            print("¡Correcto! Ahora eres su amigo. 🐉🤝")
        else:
            print(f"¡Incorrecto! El dragón se ofende porque tiene {edad_real} años y aparenta menos 😠🔥")
    
    adivina_edad_dragon()
