Ejemplo de uso de diccionarios en Python:


# Crear un diccionario
mi_diccionario = {
    "nombre": "Juan",
    "edad": 25,
    "ciudad": "Madrid"
}

# Acceder a un valor
print(mi_diccionario["nombre"])  # Salida: Juan
